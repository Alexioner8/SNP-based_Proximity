Extended Methods
In this work we applied Monopogen SNV calling pipeline and our novel SNV-based NN-cell detection algorithm to human carotid artery full-length scRNA-seq dataset of 5179 single cells generated with SMART-seq2 technology by Mocci et al. (GSE260657) Fastq files were trimmed by Trimmomatic-0.39 and aligned to human reference genome GRCh38 individually using STAR-2.7.11b package. Cell feature counts were calculated using subread-2.0.6 package. Cell-specific aligned bam files were merged and cell-tag labelled by patient (13 patients in total) excluding very low cell count patients (SRX23809081 and SRX23809084). Monopogen SNV calling pipeline as described by Dou et al. was applied to this dataset. Single-cell SNV matrix was filtered for high quality germline and somatic SNVs (Depth_ref>5, Depth_alt>5 and SVM_pos_score>0.1). Data processing was conducted in R 4.4.3 using Seurat 5.3.0 version. In brief, the raw count matrices were imported and filtered based on the following criteria: nFeature_RNA > 1000 & nFeature_RNA < 10000 & nCount_RNA > 10000 & nCount_RNA < 1000000, and percent.mt < 15. Principal component analysis (PCA) was conducted with default parameters, FindNeighbors, FindClusters (resolution=0.1) and UMAP embedding using the first 40 principal components. Marker genes for each cluster were identified using Seurat’s FindMarkers function with the wilcox test, and differential gene expression analysis was conducted on the RNA assay using the same statistical method. 
Raw files from droplet-based scRNA-seq dataset of 3882 T-cells (10X Genomics) were processed with Cell Ranger software version 7.1.0 (GSE210337). Reads were aligned to the 10X Genomics reference genome GRCh38 2024 Monopogen SNV calling pipeline as described by Dou et al.4 was applied to this aligned reads bam file. Single-cell SNV matrix was filtered for high quality SNVs (Depth_ref>5, Depth_alt>5 and SVM_pos_score>0.1). Data processing was conducted in R 4.4.2 using Seurat 5.3.0. The raw count matrices were imported and filtered based on the following criteria: nFeature_RNA > 100, nFeature_RNA < 6000, nCount_RNA > 300, and percent.mt < 15, to exclude doublets and dead cells. Principal component analysis (PCA) was conducted with default parameters, followed by CCA integration using the first 30 principal components and UMAP embedding. Cell specific TCR clonotype information was retrieved from filtered TCR-seq csv files from GSM6428652 and GSM6428654. 
Jaccard similarity index was calculated from the filtered SNV matrix (putative somatic SNVs derived from housekeeping-like features defined as SNVs present in >80% cells cluster-wise and patient-wise). Within each cell type, a shared SNV was considered present if the alternative allele was detected in at least five cells. The resulting sets of present SNVs were then compared between cell types. Similarity was quantified using the Jaccard index, defined as the size of the intersection divided by the size of the union of the two SNV sets. Jaccard values range from 0 (no shared SNVs) to 1 (identical SNV profiles).
To identify cells with highly similar SNV profiles, analyses were performed independently for each patient to avoid confounding by inter-individual genetic variation. For each patient, high-confidence SNVs were defined as variants detected in at least 80% of cells within that patient. A binary SNV matrix was then generated, where each variant was scored as present or absent in every cell.
Pairwise SNV similarity between cells was quantified by calculating the number of shared SNVs. Specifically, for every pair of cells, the number of SNVs simultaneously detected in both cells was computed, generating a cell-by-cell SNV sharing matrix. Self-comparisons were excluded from downstream analyses.
For each cell, the maximum number of shared SNVs observed with any other cell was determined. To focus on the most confidently related cells, only cells whose maximum shared-SNV count fell within the top 1% of the patient-specific distribution (99th percentile) were retained. This filtering step enriched for cell pairs exhibiting exceptionally high SNV similarity relative to the remainder of the population.
Source and nearest-neighbor cells were annotated according to their transcriptionally defined cell type identities. Cell-type transitions were then summarized by counting the number of nearest-neighbor relationships connecting each source cell type to each target cell type. To visualize patterns of SNV similarity across cellular populations, nearest-neighbor relationships were aggregated across all patients and displayed as alluvial plots.
Differential expression analysis was performed between macrophage populations defined by SNV-based nearest-neighbor relationships. Macrophages whose closest SNV-based nearest neighbor belonged to the macrophage population were classified as bona fide macrophages (NN-MPs_MPs), whereas macrophages whose closest SNV-based nearest neighbor belonged to the smooth muscle cell (SMC) population were classified as SMC-adjacent macrophages (NN-SMCs_MPs). Differential gene expression between NN-SMCs_MPs and NN-MPs_MPs was assessed using the Seurat FindMarkers framework. Analyses were performed using both the MAST hurdle model and the Wilcoxon rank-sum test to evaluate the robustness of identified transcriptional differences.

Detailed Scripts for reproducing analysis and QC metrics are available at https://github.com/Alexioner8/SNP-based_Proximity.
